# Nebulosa-Mobile

## Instalar projeto

```bash
yarn install
```

## Rodando o projeto

```bash
expo start || yarn start
```

Instalar o aplicativo da [EXPO](https://play.google.com/store/apps/details?id=host.exp.exponent) em seu smartphone, ler o qr code gerado apos rodar o projeto.
